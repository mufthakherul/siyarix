# Contributing

We welcome contributions. Please follow these steps:

1. Fork the repo and create a feature branch.
2. Run tests locally and add unit tests for new features.
3. Keep commits small and use conventional commit messages (feat, fix, docs, chore).
4. Open a pull request against `main` and include a short description and tests.

Code of conduct:
- Follow standard respectful behavior and label security-sensitive issues as private when needed.
