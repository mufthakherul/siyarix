# Security Policy

We take security seriously. If you discover a vulnerability, please follow the responsible disclosure process below.

1. Report the issue to the project maintainers via email: security@cosmicsec-lab.example (replace with official address) or create a private GitHub security advisory.
2. Provide a detailed description, steps to reproduce, affected versions, and any PoC you can share.
3. We will acknowledge receipt within 48 hours and provide a remediation timeline.

Do NOT post security-sensitive details publicly until a fix or mitigation is available.
