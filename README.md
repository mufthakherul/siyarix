# Siyarix

AI-native cybersecurity operations platform providing autonomous execution, multi-agent framework, intelligent workflow orchestration, and advanced security tooling for offensive and defensive operations.

## Installation

```bash
pip install siyarix
```

For all capabilities (AI, CLI, SIEM connectors):

```bash
pip install "siyarix[all,cli,siem]"
```

## Quick Start

```bash
siyarix --help
```
