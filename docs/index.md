# Welcome to Siyarix v3

**Siyarix** is an AI-native cybersecurity operations platform that acts as your personal AI orchestration agent. It bridges the gap between natural language security objectives and deterministic tool execution. 

Siyarix takes natural-language requests—like *"scan this subnet for open ports"*—routes them through a robust multi-provider AI abstraction layer, generates an execution plan, and runs local open-source tools to deliver precise results safely.

---

## Why Siyarix?

!!! tip "Human oversight with machine speed"
    We believe AI shouldn't replace security operators, but it should take over the repetitive scripting, tool parsing, and workflow glue. 

Siyarix is built with three core design pillars:
- **CLI-First**: Everything works right in your terminal. No web dashboards needed.
- **Provider Agnostic**: Use OpenAI, Anthropic, Gemini, or even local models via Ollama. If one fails, it automatically fails over.
- **Safe & Deterministic**: Every command generated by AI is blocked by default until it passes syntax and danger analysis gates. 

---

## Capabilities at a Glance

- **50+ Commands**: Ranging from basic recon and vulnerability scanning to complex playbook executions.
- **Interactive Chat REPL**: Context-aware terminal UI with full slash-command support.
- **Multi-Wave Execution**: Iterative LLM reasoning for complex tasks.
- **Over 100+ Integrated Tools**: Siyarix automatically discovers tools on your `PATH` (like `nmap`, `nuclei`, `metasploit`) and handles parsing their outputs.
- **Compliance & Threat Intel**: Maps findings to MITRE ATT&CK, handles STIX feeds, and assesses against SOC2, ISO27001, and more.
